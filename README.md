# vue-component-generator
> A generator for vue component


## Install

```bash
npm install vue-component-generator -g
```

## Usage

### Init Component

Run command:
```bash
vcg <component-name>
```
.eg
```bash
vcg component-demo
```

### Install Depandencies

```bash
npm install
```

### Start development

```bash
npm run dev
```

Open `localhost:8877` in your browser

### Run test

```bash
npm run test
```

### Build Component
```bash
npm run build
```
